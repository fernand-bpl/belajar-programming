# Minimal Python starter

This is a minimal Python starter included in the repository for quick experiments.

Files added:
- `src/app.py` — contains `greet(name)` and a small CLI.
- `tests/test_app.py` — simple unittest for `greet`.

How to run:

1. (Optional) create and activate a virtual environment.
2. Install dependencies (none required for this tiny example).
3. Run tests:

```powershell
python -m unittest discover -s tests
```

Run the CLI:

```powershell
python src\app.py Alice
```
